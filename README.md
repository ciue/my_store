# my_store
- 基于Vue2.0、Vuex、Vue-router实现的商城系统

## 使用的库/工具
- vue-cli (webpack脚手架)
- vue-router(地址页路由跳转)
- vuex(地址页状态管理)
- axios(数据请求)
- mint-ui(无限滚动懒加载)
- hammerjs(手势判断)
- swiper(轮播图)
- RAP2.0(模拟后台数据)


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
